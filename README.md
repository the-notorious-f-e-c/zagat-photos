## How To Use

```bash
# clone this repository
$ git clone https://github.com/the-nuggets/photos-and-modal.git

# install dependencies
$ npm install

# seed database
$ npm run seed

# compile/transpile files with webpack
$ npm run build

# run the app!
$ npm run start
```
